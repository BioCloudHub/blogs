# 关于 BioCloudHub

BioCloudHub 是一个面向生物制药行业的技术博客站点，专注于把研发知识转化为可执行的工程实践。

## 关注方向

- CMC：细胞株开发、工艺优化、质量研究、法规与申报。
- 生物分析：基因组学、蛋白质组学、单细胞数据分析。
- 药物研发：AI 分子设计、虚拟筛选、临床统计分析。
- 计算工程：云原生平台、容器编排、自动化交付。

## 内容原则

- 以“可落地”为核心，优先给出方法、参数与流程。
- 兼顾科学性与工程性，连接研发与生产两端。
- 持续迭代专题内容，构建长期可维护知识库。
