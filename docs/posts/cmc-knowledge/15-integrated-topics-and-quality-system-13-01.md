## Q601. 二次干燥

:::answer

二次干燥,也称解吸干燥,是指在初级干燥除去冰(自由水)后,在二次干燥过程中除去未冻结的结合水(吸附在干燥的滤饼表面)。 在初级干燥结束时,搁板温度会随着时间的增加而升高,应注意不要使搁板温度升高过快而使冻干制品塌陷．此外,应保持解吸压力与初级干燥时的压力相同．虽然一些研究报道了在冻干设备可以达到的最低压力下进行二次干燥,但是没有任何证据证明最低压力可以加速脱附过程。 另外,只有在初级干燥完成后才应开始二次干燥,因此,确定初级干燥结束的时间尤为重要．在二次干燥过程中,冻干药品的最终残余水分需要精确控制,因为许多产品可能会被过度干燥所破坏,从而导致储存时药物活性的损失,以及随着时间的推移,残余的水分会加速药物降解．在大多数情况下,二次干燥后需要将水分控制在 1%以下。

:::

## Q602. N 端序列分析的重要性

:::answer

蛋白质的 N 端序列是指蛋白质链的起始部分，通常由一串氨基酸残基组成。这个起始片段对于蛋白质的功能和定位至关重要。通过对蛋白质 N 端序列的分析，我们可以揭示蛋白质的起始氨基酸、信号肽、转运蛋白等关键信息，这些信息对于了解蛋白质的功能及其在细胞中的位置非常有价值。 此外，蛋白质的生物合成起始于 N 端，其 N 端序列的组成对生物学功能有着巨大的影响，例如蛋白质的半衰期、蛋白质在亚细胞器中的定位等等。并且 N 端可发生多种翻译后修饰，这些也与蛋白质的功能和稳定性息息相关。因此，对蛋白质 N 端序列的分析，有利于帮助科研工作者分析蛋白质的高级结构，揭示蛋白质的生物学功能。 < 抗体药物结构表征 >

:::

## Q607. 什么样的样品不适合用 Edman 降解法 N 端测序来分析序列？

:::answer

- **第，N 末端封闭的样品：** 从反应原理可知，N 末端的 α-氨基被修饰后，反应无法进行； 第二，关注的 N 端序列中有太多非标准氨基酸，由于没有相应的标准品，也没有办法通过 N 端测序来分析。此时可改用质谱法进行序列确证。

:::

## Q608. 影响 Edman 降解结果的因素

:::answer

测序使用的化学试剂如果纯度不够高，也有可能在 PTH 色谱中产生假峰，从而影响对氨基酸序列 的准确判定。 由于各种氨基酸残基化学或物理性质不同，因此判断其含量时 PTH 信号的强弱会发生变化。例如，酪氨酸和丝氨酸容易发生脱羟基造成峰值发生变化，当半胱氨酸发生修饰时很难检测到信号，组氨酸和精氨酸由于极性难以被萃取，甲硫氨酸容易被氧化，这些因素都会导致峰值相应较低。 带有修饰残基在检测时表现出与其他未修饰氨基酸相似的峰也会造成氨基酸识别错误。

:::

## Q610. N 端测序其它方法

:::answer

除了 Edman 降解法，质谱技术（尤其是串联质谱）也被广泛用于抗体蛋白质的氨基酸序列分析，质谱法将抗体蛋白消化成 5-25 个氨基酸的肽，随即通过 LC-MS/MS 分析检测，将采集的数据与理论序列数据库进行匹配，进而确认 N 端序列。质谱可以测出封闭和修饰的抗体蛋白药物 N 末端，且已成为首选的全长抗体测序方法。 Edman 降解法与质谱法不同，其不需要借助蛋白数据库便可以直接进行序列分析。但它不能用于 N-末端封闭或修饰后的氨基酸残基。

:::

## Q612. 什么是因子间的交互作用

:::answer

- **定义两因子间的交互作用为：** 如果因子 A 的效应依赖于因子 B 所处的水平时，则称 A 与 B 之间有交 互作用。 因子的交互作用指的是两个或多个因子相互影响，导致其效果不同于它们单独作用的情况。例如，假设一个实验中有两个因子,一个是肥料的种类，另一个是浇水的频率。当这两个因子同时作用时，可能会发生肥料种类和浇水频率交互作用的效应，导致植物的生长出现意想不到的结果。因子的交互作用在科学研究和现实生活中都有广泛的应用。在医学研究中，不同的药物、剂量和 疗程之间可能会产生交互作用，从而影响疾病治疗的效果。

:::

## Q614. KPP 的理解

:::answer

- **重要工艺参数(KPP)：** 产品开发试验或是生产过程中表明对工艺性能产生影响的工艺参数可将其归类为重要工艺参数(KPP)。重要工艺参数可能会影响工艺性能属性(如,细胞培养过程中的抗体滴定度或下游纯化中的产量),但不会影响产品关键质量属性(CQA)。 通过 PC 研究实验结果确定 KPP。

:::

## Q615. 什么是肽段覆盖率？

:::answer

肽段覆盖率是指检测到的肽段氨基酸数量占该蛋白质总氨基酸数量的比例。

:::

## Q616. 肽段覆盖率分析的意义

:::answer

蛋白质供试品肽段覆盖率的检测，对于蛋白质类药物的一级氨基酸序列的确证，保证蛋白质类药物的高级结构形成及维持蛋白质类药物性质均具有很重要的意义。

## Q618. 酶切

:::answer

蛋白样品鉴定过程一般只会使用 Trypsin 对蛋白进行酶切，鉴定到的肽段覆盖率大概在 60%。 为了得到靶蛋白的全部序列信息，会选用在蛋白鉴定过程中常用的 6 种蛋白酶（Trypsin、Chymotrypsin、Asp-N、Glu-C、Lys-C 和 Lys-N）分别对靶蛋白进行酶切和鉴定，在得到更多的碎裂肽段片段的同时，经过肽段之间的拼接完成蛋白序列 100%的测定。

:::

## Q619. 肽段覆盖率实验样品要求

:::answer

对样品纯度没有太高的要求，但是从原则上讲，样品纯度越高，同样上样量的前提下，肽段覆盖率越高；需要的靶蛋白样品量：10-20μg 即可。

:::

## Q620. 如何能得到更高的肽段覆盖率？

:::answer

在尽量保证蛋白纯度的前提下，样品前处理时，根据蛋白理论氨基酸序列结合质谱级别蛋白酶酶切位点信息使用特定蛋白酶对蛋白进行酶切，尽量选择酶切位点分布均匀的蛋白酶酶切，提高酶切效率，保证酶切之后的肽段混合物长度适宜离子化，以便获得更高的覆盖度。

:::

## Q622. 抗体药物原液储存方式

:::answer

蛋白质药物原液通常采用冷冻技术，通过延长药物原液的保质期以及提供更好的生化稳定性来实现生产灵活性。

:::

## Q623. 抗体药物原液冷冻的优势

:::answer

1）冷冻储存可以通过最大限度地降低分子迁移率以及与容器封闭表面的相互作用来降低物理和化学降解的速度，从而减缓反应动力学。 2）由于冷冻溶液条件不利于微生物生长，所以可降低微生物污染的风险。3）最大限度地减少运输过程中搅拌/震动引起的降解的潜在风险。4）冷冻原液更长的储存时间为满足临床或商业化药物产品生产的实时需求提供了灵活性，也降低了药物开发的总体成本。

:::

## Q624. 原液冷冻蛋白质浓度研究

:::answer

通常蛋白质的浓度也是影响稳定性重要因素，如多亚基的蛋白质和辅助因子容易解离、聚集、沉淀等。因此在储存过程中需要研究合适的蛋白质浓度保存浓度。

:::

## Q625. 原液冷冻保护剂的添加

:::answer

蛋白质一般在疏水环境中才能长期保存，加入某些稳定剂可以降低蛋白质溶液的极性，以免变性失活，这类保护剂有渗透物稳定剂及离子稳定剂，渗透物稳定剂的主要作用是影响水的黏性和表面张力，包括糖类、多元醇、多糖、中性多聚物、氨基酸及其衍生物，以及较大的偶极分子（如三甲胺-N-氧化物），离子稳定剂主要是通过加强分子内接触来促进水溶液中蛋白质的有序或折叠构象，如盐类，保护剂的种类。

:::
