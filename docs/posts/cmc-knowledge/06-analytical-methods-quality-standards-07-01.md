## Q265. 对 HCP 的理解

:::answer

## Q266. HCP 的监管要求

:::answer

各指导原则或法规中，都明确规定 HCP 要控制在可接受水平。当下对宿主蛋白残留风险的评估具有较多的不确定性，缺乏广泛适用的标准以及方法。因此欧洲及美国药监局对宿主蛋白残留量的限量要求也是依据个案原则，不设统一的标准。并且建议根据工艺特点，药品应用剂量，给药方式，用药频率，临床前以及临床研究数据，对宿主蛋白的质量风险进行质量风险综合评估。借此对相应产品工艺进行修订，以期降低宿主蛋白残留的质量风险。 美国和欧洲药典的监管指南并未充分指明生物制药行业必须严格遵守的具体可接受的 HCP 水平，但强调了它们的重要性和尽量减少 HCP 杂质的必要性。一般来说，行业标准的目标是在最终药品中达到 100ppm 或更少。如下： FDA 审查的大多数生物技术产品含有 1-100ppm 的宿主细胞蛋白(HCP)杂质。中国药典如下： 重组乙型肝炎疫苗(汉逊酵母)，应不超过总蛋白含量的 1.0%； 重组乙型肝炎疫苗(CHO 细胞)，应不超过总蛋白含量的 0.05%； Sabin 株脊髓灰质炎灭活疫苗(Vero 细胞)，应不高 200ng/剂； 注射用人促红素(CHO 细胞)，应不超过总蛋白含量的 0.05%。

:::

## Q267. HCP 的去除工艺

:::answer

- **通常通过以下纯化步骤会降低 HCP 水平：** 1）深层过滤 2）亲和层析 3）离子交换层析 4）混合层析

:::

## Q268. 深层过滤去除 HCP

:::answer

深层过滤，无论是单独的还是和离心结合使用，已经被广泛的应用于收获澄清。许多互相补充的分离机制涉及其中。除了通过机筛截留细胞碎片/大颗粒，带负电的深层滤器可以通过吸附作用来捕获 可溶解的亚微颗粒。因此，深层滤器致力于整体的 HCP 去除。HCP 通过静电和疏水作用机制去除。 值得注意的是，上样发酵液或离心后的收获液对 HCP 的有效去除有很大的影响。一旦达到饱和状态，随着载量增加 HCP 将会流穿。多阶段深层过滤（通过一连串的深层滤器过滤）能更好的去除 HCP。在滤器选择和放大评估时，深层滤器去除 HCP 的能力经常被忽略。强烈推荐，在选择一个滤器和确定其操作参数时，除了上样载量，也要评估和考虑 HCP 去除载量。

:::

## Q269. 亲和层析去除 HCP

:::answer

亲和层析通常被用于抗体的捕获。由于其高度选择性，通常亲和层析一步可以去除>90%的 HCP，是下游工艺中最有效的去除 HCP 的单元操作。 为尽量减少 ProteinA 洗脱液中 HCP 的水平，通常在目标产物洗脱之前使用介于平衡和洗脱缓冲液之间的 pH 进行中间淋洗。ProteinA 洗液中 HCP 的残留主要是通过与抗体之间的相互作用，而不是与填料直接相互作用的。这表明，为了最大限度去除 HCP，中间淋洗步骤优化的重点在于破坏 HCP 与抗体之间的相互作用，而不是 HCP 与 ProteinA 之间的相互作用。在工艺开发阶段，首先应选择合适的 ProteinA 亲和层析填料，这样能够在一定程度上降低 HCP 与填料之间非特异性吸附。使用中间 pH 缓冲液进行淋洗的方式只能去除与亲和填料结合较弱的 HCP，不足以打破 HCP 与抗体之间的相互作用。使用低 pH 进行淋洗，尤其是在高载量下，可能会因抗体被部分洗脱，造成收率损失。因此，我们可对中间淋洗步骤缓冲液盐的种类、浓度、pH 等条件进行优化，还可以添加一定量的盐（NaCl）或者添加剂[2]，如精氨酸、尿素、盐酸胍等，有效地破坏 HCP 与抗体间的相互作用，降低洗脱液中 HCP 的水平

:::

## Q270. 离子交换层析去除 HCP

:::answer

虽然 ProteinA 步骤能去除 90%以上的 HCP，若仍无法达到要求，还需要考虑精纯步骤进行优化去除，常用离子交换层析法。 1）阴离子交换层析：大多数抗体的等电点为 6.5-9.0，HCP 等杂质的等电点通常呈现酸性。在低于抗体等电点的 pH 和较低的电导下，抗体从阴离子层析柱中流穿，而 HCP 等杂质与填料结合，在抗体纯化工艺中，阴离子层析通常采用流穿模式。 2）阳离子交换层析：阳离子交换层析结合带有正电荷的蛋白质，通过增加盐浓度或 pH 将蛋白洗脱下来。由于抗体在中性缓冲液条件下一般带正电，因此在抗体生产工艺中阳离子交换步骤采用结合洗脱模式去除产品相关的杂质，主要去除 HCP、rProteinA 和聚集体。

:::

## Q271. 混合层析去除 HCP

:::answer

抗体表面疏水基团和带电荷的分布与大多数 HCPs 表面分布不同，因此混合模式的填料可在去除 HCP 上显现出高于单一模式填料的选择性和特异性。 混合模式层析依赖于填料与离子和疏水相互作用产生的精细结合和洗脱条件的相互作用，以及氢键的影响。因此，pH 主要控制离子相互作用和结合能力，有利于高回收率，而电导率(即盐浓度)决定了支持分离能力的疏水相互作用，因此这两个参数的优化至关重要。此外，添加剂[10]的使用也可以会提高其的分离性能和选择性。

:::

## Q272. HCP 形成的原因

:::answer

细胞培养液中的 HCP 水平取决于细胞培养工艺和收获条件。 研究表明，从死亡宿主细胞中释放的 chromatin 是聚体形成和宿主细胞污染来源的主要原因。chromatin 主要由 DNA 和组蛋白构成，两者都具有易于和非组蛋白 HCP，单抗结合的高活性表面。chromatin 主要以复杂的多聚体形式存在，该多聚体由 DNA，组蛋白核心以及非组蛋白 HCP，单抗组成，与单抗相比，该多聚体结合 proteinA 树脂更坚固。洗脱过程中，结合在 proteinA 上的 chromatin 聚体 leachates 也是洗脱液中 HCP 污染主要来源。

:::

## Q273. 游如何降低 HCP 的水平

:::answer

- **如下的文献报道：** Valente 等人研究了在相同条件下培养的不同年龄的 CHO-K1 细胞，并显示 92 种 HCP 的差异表达，其中 34 种先前被报道难以通过纯化去除，17 种已知与 mAb 强烈相互作用。批次培养和补料分批培养之间的选择也在 HCP 组成中发挥着重要作用。补料分批培养涉及间歇性补给特定营养素，以避免关键培养基成分耗尽，从而延长培养和 mAb 生产的寿命。这种提高细胞生长和生产力的方法会导致更高水平的 HCP，这些 HCP 从死细胞中释放出来，也会从活细胞中分泌出来，其在细胞外的积累比批次培养中多，可能会损害产 品质量。 Park 等人使用蛋白质组学分析表明，与批次培养相比，在补料分批培养中鉴定和定量的 HCP 明显更多。虽然通过灌流生物反应器进行连续培养可以提高 mAb 的生产力，但需要从治疗产品中去除的细胞外 CHOHCP 杂质的组成也可能在延长的细胞培养过程中发生变化，并在 DSP 操作期间挑战去除。 Goey 等人观察了细胞培养温度对 CHO 细胞行为和细胞外 HCP 谱的影响。在对数细胞生长期晚期使用温和低温条件减少了细胞内和细胞膜 HCP 的数量，包括大约 44%的蛋白酶和 27%的伴侣物质，尽管 HCP 浓度与对照培养相似。在亚低温下抑制凋亡细胞群后，许

## Q278. HCP 介绍

:::answer

重组蛋白质药物的生产需选用合适的宿主细胞，而宿主细胞在表达目标产物时将不可避免地合成多种 HCP，这些蛋白会对药物造成污染可能影响药物的活性、药效以及安全性。不仅如此，HCP 作为引人人体的外源蛋白质，其免疫原性可能诱导人体产生抗体，引起机体免疫反应。同时,HCP 潜在的佐剂效应也可能会造成机体对药物进行免疫应答，进而影响药物的药效，缩短其 半衰期。由此可见，重组蛋白质药物中残余。

:::

## Q279. HCP 检测

:::answer

- **ELISA(酶联免疫吸附测定法)。（1）原理：** 特异性抗 HCP 抗体被固定在酶标板上。待测样品中的 HCP 与抗体结合，然后再加入与 HCP 结合的酶标记的二抗。通过测定酶的活性，可以间接测量 HCP 的含量。（2）优点：灵敏，可以量化 HCP 的总量。（3）缺点：需要高质量的抗 HCP 抗体，可能不检测到所有的 HCP 杂质。 westernblot 法。westernblot 法检测 HCP 具有较强的免疫特异性，能够分离并分析多种组分，但该检测方法仅能进行限量分析，且灵敏度较低，对于操作的要求也较高，往往不适用于生产质控中。

:::

## Q280. DNA 检测

:::answer

在《中国药典》三部 2010 年版中，检测残余 DNA 的方法推荐为地高辛标记探针杂交法和荧光染色法。地高辛标记探针杂交法常见于膜斑点杂交法。该方法采用地高辛标记标准细胞基因组 DNA 片段，制成特异性探针，通过与待检品进行同源 DNA 杂交，检测制品中相应的细胞 DNA 残余。在膜斑点杂交法中，制成地高辛标记的探针是该方法的关键。 实时荧光定量 PCR（qPCR）法也被用于进行生物制品中生产用细胞相关的 DNA 片段分布的分析。检测宿主细胞残留 DNA 含量的方法都相对成熟，而且药典法规也有明确规定，在《中华人民共和国药典》2020 年版第三部中也推荐使用定量 PCR 法。qPCR 法相比于 CCD 和 CGE-LIF，有操作更简单，耗时更短，且与宿主细胞残留 DNA 含量检测值具有很好的一致性和可对比性等优势。

:::

## Q281. proteinA 检测

:::answer

proteinA 种类比较单一，并且其与抗体分子具有较强的结合力,ELISA 法自然成为检测 proteinA 残留的首选。目前国内多采用商品化的 ProteinA 检测试剂盒，操作方便，结果较精确。但在选用试剂盒检测 proteinA 残余时，部分 proteinA 仍结合着药物蛋白。由于位阻效应的影响，结合了药物蛋白的 proteinA 不能与试剂盒中的 proteinA 捕获抗体结合，同时，结合在 proteinA 上的药物蛋白会阻碍酶标抗体的结合，造成 ProteinA 残余的漏检。因此，选用商品化的 proteinA 检测试剂盒，尚需排除药物蛋白对检测的干扰，并进行干扰试验。

:::

## Q282. 抗体蛋白结构表征项目

:::answer

蛋白质是由不同氨基酸连接形成的多聚体，并且通过正确折叠为一个特定构型，发挥蛋白药物的生物学功能。氨基酸序列的特定位置可以与化学基团共价结合，发生蛋白质翻译后修饰，这些翻译后修饰会导致蛋白的结构发生改变，从而影响蛋白药物的生物学活性，所以需要对蛋白的分子量、肽段覆盖率、翻译后修饰等进行检测。本文汇总结构表征项目如下： 01-一级结构表征 02-高级结构表征 03-蛋白异质性表征 04-杂质检测

:::

## Q283. 一级结构表征~分子量

:::answer

- **项目介绍：** 分子量的检测是鉴定蛋白的第一步，使用高分辨质谱分析可得到蛋白质的多电荷信号，通过对信号进行去卷积分析，可获得精确分子量数值，并初步判断蛋白的修饰状态。对于抗体药物还可打开轻重链或者去除糖基，分别分析糖基化和去糖基化轻链和重链的分子量。 检测方法：液相色谱-质谱联用(LC-MS)是一种结合液相色谱(LC)和质谱(MS)的强大分析技术，其允许分离、检测和量化混合物中的各种化合物。LC-MS 中的高分辨质谱是一种能够更高精准度测量分子量的质谱技术，这使得该技术尤其适用于复杂的、分子量相近的大分子化合物检 测。 < 生物技术药物糖基化修饰分析及控制 >

:::

## Q284. 一级结构表征~N 端序列分析

:::answer

- **项目介绍：** 蛋白质的生物合成起始于 N 端，其 N 端序列的组成对生物学功能有着巨大的影响，例如蛋白质的半衰期、蛋白质在亚细胞器中的定位等等。并且 N 端可发生多种翻译后修饰，这些也与蛋白质的功能和稳定性息息相关。因此，对于蛋白质 N 端序列的分析，有利于帮助科研工作者分析蛋白质的高级结构，揭示蛋白质的生物学功能。 检测方法：Edman 降解法 N 端测序是分析这一属性的常规方法。通过 Edman 降解法分析 N 端的 15 个氨基酸序列在生物药申报时通常是必检项目；同时也是很多已上市生物药的年检项目（2020 版中国药典《人用重组 DNA 蛋白制品总论》）。

:::

## Q285. 一级结构表征~C 端序列分析

:::answer

- **项目介绍：** C 端序列是蛋白质和多肽的重要结构与功能部位,对蛋白质的生物功能甚至起决定性作用。此外，由于蛋白翻译后在细胞内需要进一步剪切和修饰，通常不能简单的使用基因序列对 C-端或 N 端做简单的预测，对于蛋白生物制品来说，使用 C-端测序必不可少，随着蛋白质组学研究的不断深入，蛋白质 C 端测序对其功能研究将发挥越来越重要的作用 检测方法：蛋白质 C 端测序的方法主要有羧肽酶法、化学法和串联质谱法等。羧肽酶法和化学法的策略是通过酶解法或化学试剂裂解法获得 C 端氨基酸，再结合液相色谱或质谱技术对氨基酸种类进行鉴定。串联质谱法测定蛋白质 C 端序列的原理是将蛋白质样品酶解、消化成肽段混合物，然后对肽段混合物进行串联质谱分析，通过一级质谱信息选择 C 端肽段离子进行二级质谱分析，再根据二级质谱数据结合相应软件和数据库计算蛋白质的 C 端氨基酸序列。

:::

## Q286. 一级结构表征~抗体 C 端 K 缺失比例分析

:::answer

- **项目介绍：** 抗体类蛋白质药物分子重链 C 末端赖氨酸会有不同程度的缺失现象发生，质量控制中需要建立对这类现象的定性和定量分析的方法。C 端赖氨酸(K)缺失后，会造成供试品酶解后 C 末端多肽分子的序列发生变化，并且这两种 C 末端多肽分子（缺失或者不缺失 K）的定量结果可以反映蛋白质分子 C 端赖氨酸的缺失程度。N-和 C-末端测序是非常重要的质量控制分析，并且在 ICHQ6B 生物技术产品测试程序指南中 C 端 K 缺失测定是强制性的。 检测方法：IEF、cIEF、离子交换色谱和 LC-MS 等方法常用于分析 C-末端赖氨酸的变异。目前，将样品进行胰酶消化，再通过 LC-MS 对样品进行分析，是目前检测抗体 C 端 K 缺失比例比较成熟普遍的方式。

:::
