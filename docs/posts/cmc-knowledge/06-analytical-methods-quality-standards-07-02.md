## Q287. 一级结构表征~肽段覆盖率

:::answer

- **项目介绍：** 肽段覆盖率是指检测到的肽段氨基酸数量占该蛋白质总氨基酸数量的比例。蛋白质供试品肽段覆盖率的检测，对于蛋白质类药物的一级氨基酸序列的确证，保证蛋白质类药物的高级结构形成及维持蛋白质类药物性质均具有很重要的意义。目前，对蛋白质肽段覆盖率的检测根据药物申报要求的规定，均采用质谱法进行检测。 检测方法：将样品通过酶切，进入质谱采集肽段的一级质谱信息和二级碎片离子质谱信息。采用软件对总离子流图进行分析，将测得肽段的相对分子量与理论值进行比较，并且通过二级质谱离 子确认多肽的序列正确性。根据已确认肽段的氨基酸序列与理论序列的比值计算肽图覆盖率。

:::

## Q288. 一级结构表征~肽图分析

:::answer

- **项目介绍：** 同一蛋白在相同条件下得到的肽图是基本相同的，而不同的蛋白由于一级结构不同，由 Lys-C 酶水解得到的肽段存在差异，因此肽图也存在差异。通过对比样品与参比品的肽图，可判断样品是否与参比品具有相同的一级结构，即鉴定蛋白质一级结构的完整性和准确性。因此，肽图分析对多肽结构研究和特性鉴别具有重要意义。通过对肽图的分析，可以验证蛋白质的序列和修饰情况，揭示蛋白质的结构特征和变化。这对于生物药物的研发过程中的结构确认和质量控制具有关键意义。 检测方法：现行版中国药典四部中明确了胰蛋白酶裂解-反相高效液相色谱法和溴化氰裂解法。近年来，随着液相色谱技术的发展，反相液相色谱（RP-UPLC）成为多肽分离鉴定的主要工具。用 Lys-C 酶将蛋白质酶解成特定的肽段后，进入超高效液相色谱分离，通过对特征峰的分析，确认样品与参考品一级结构的一致性。

:::

## Q289. 一级结构表征~氨基酸组成分析

:::answer

- **项目介绍：** 氨基酸组成分析是准确定量多肽、蛋白质、抗体等样品的黄金标准。通常我们希望能在找到样品的序列信息之前先知道样品中的氨基酸组成，因为这有助于我们发现测序过程中是否存在错误，以对测序流程进行修正。了解样品中氨基酸的组成，还可以协助判断哪种蛋白酶适用于样品的水解过程，此外还可以确定蛋白质中低水平非标准氨基酸（例如正亮氨酸）的掺入。确定氨基酸组成的通用方法通常被称为氨基酸分析。可用于确定氨基酸、肽、多肽、蛋白质和抗体 样品的氨基酸组成，成分，含量等分析。氨基酸分析也称为氨基酸成分分析，氨基酸组成分析。 能够定性以及定量的对样品中的氨基酸组成进行测定。 检测方法：目前氨基酸分析的方法主要包括离子交换色谱法（IEC）、高效液相色谱法（HPLC）以及离子色谱法（IC）等。这三种方法都包括分离-衍生-检测的步骤：IEC 通过阳离子交换柱分离，柱后茚三酮衍生，利用光度法进行测定；HPLC 通过反相色谱分离，柱前衍生，进行荧光或 紫外检测；IC 利用阴离子交换分离，安培法直接检测

:::

## Q290. 一级结构表征~消光系数测定

:::answer

- **项目介绍：** 消光系数是衡量化学物质吸收特定波长光强度的度量单位。蛋白质在 280nm 的摩尔消光系数几乎完全取决于芳香族残基的数量，尤其是色氨酸，并且可以通过氨基酸序列预测。若摩尔消光系数已知，则可用于确定溶液中蛋白质的浓度。 检测方法：常见的可用于测定消光系数的方法是分光光度法，这也是目前大多数生物制品测定消 光系数采用的方法。

:::

## Q291. 一级结构表征~寡核苷酸序列分析

:::answer

- **项目介绍：** 寡核苷酸序列分析是一种用于研究和确定短的 DNA 或 RNA 片段（通常为 20-30 个核苷酸）核苷酸种类和排列顺序的生物技术，在许多生物学和医学领域都有重要的应用价值。寡核苷酸是药物研究中的一个新热点，一些具有特异性结合或者裂解致病基因的寡核苷酸可以开发成新型药物，它们作用效率高、应用范围广，可以对传统药物进行补充，具有广泛的应用前景。对寡聚核苷酸进行序列分析是基因药物的研发和生产过程中必不可少的一环。 检测方法：质谱技术的发展为寡核苷酸序列鉴定提供了强有力的分析方法。寡核苷酸分子在高分辨率质谱仪中被解离成一系列碎片离子，这些碎片带有质荷比（m/z）值，通过对这些片段离子的质谱分析，可以推断原始寡核苷酸的碱基序列。

:::

## Q292. 高级结构表征~二硫键

:::answer

- **项目介绍：** 二硫键（S-S 键）是通过蛋白质中两个半胱氨酸上的巯基(-SH)氧化而形成的，是重要的蛋白质翻译后修饰形式。蛋白质通过各种链间和链内的半胱氨酸连接在一起，这对蛋白质分子保持正确的高级结构，维持蛋白质的生物活性至关重要。抗体药物中二硫键的排布是对药物结构特性的一种反映，因此二硫键连接形式的确认成为抗体药物结构确认过程中具有非常重要的作 用。检测方法：检测和鉴定二硫键有多种方法，以下列举了一些常用的技术。质谱法:在质谱中，蛋白 质首先被断裂成较小的片段。通过检测这些片段并对其进行数据分析，可以确定硫桥连接的半胱 氨酸残基的位置。Ellman's 试剂:该试剂可以与游离的半胱氨酸及还原的二硫键反应，产生黄色的产物，其吸光度可以在 412nm 处进行测量。通过比较样品在不同条件下的吸光度，可以估计二硫键的数量。荧光标记法:某些荧光试剂可以与游离的半胱氨酸残基反应，但不与形成二硫键的半胱氨酸反应。这种方法可以用来估计游离半胱氨酸的数量

:::

## Q293. 高级结构表征~圆二色性分析

:::answer

- **项目介绍：** 圆二色谱仪利用蛋白质的圆二色性以及对左右圆偏振光吸收的不对称性进行结构分析。蛋白质或多肽中的主要光活性基团包括肽键、芳香氨基酸残基和二硫键等。当平面圆偏振光的吸收不对称时，就会产生吸收差值，这种差异引起偏振光矢量的振幅差，使圆偏振光变成了椭圆偏振光，这就是蛋白质的圆二色性。通过使用圆二色谱扫描仪测量蛋白质等生物大分子的圆二色光谱，可以简单而快速地鉴定生物大分子的二级结构，广泛应用于蛋白质折叠、构象研究和酶动力学等领域。 检测方法：圆二色谱是应用最为广泛的蛋白质二级结构测定方法，也是研究稀溶液中蛋白质构象的一种快速、简单且相对准确的方法。它可以在溶液状态下进行测定，更贴近蛋白质的生理状态。圆二色谱测定方法快速简便，对构象变化灵敏，因此成为研究蛋白质二级结构的主要工具之一，并在蛋白质构象研究领域得到广泛应用。

:::

## Q294. 高级结构表征~自由巯基分析

:::answer

- **项目介绍：** 自由巯基非常活泼，可以与其他蛋白质结合而改变其结构和功能，也可以与氧分子反应产生氧化物损害细胞和组织。自由巯基定量是抗体等蛋白质类生物制品的一个重要检测项目。例如，含有自由巯基的抗体的生物学功能有时比完整抗体会有所下降，甚至差距很大。因此，快速响应并精确定量蛋白质类生物制品中的自由巯基，对产品的早期工艺开发和后期生产监控都至关重要。 检测方法：Ellman 法是检测自由巯基浓度的常规方法，原理为 DTNB 试剂与游离巯基反应生成 2-硝基-5-硫代苯甲酸（TNB-），若在中性或碱性 pH 条件下的水中可以离子化，生成 TNB2-二价阴离子。这种 TNB2-离子呈现黄色，通过测定在 412nm 可见光吸光度就可以定量 TNB2-，然后可以用半胱氨酸标准品定量巯基或者基于摩尔吸收率定量巯基的方法对自由巯基进行定量。

:::

## Q295. 高级结构表征~颗粒的尺寸分布

:::answer

- **项目介绍：** 确定溶液样品中悬浮体或聚合物中颗粒尺寸和半径分布，是其质量控制中的主要内容之一。 检测方法：DLS 分析是一种非侵入性的光谱分析技术，用于研究和分析物质的光散射特性。它是一种非常有用的工具，可以提供关于样品中粒子的大小、形状、浓度和分布等信息。DLS 分析是通过测量光在样品中发生散射的方式来获得信息。当光线通过样品时，它会与样品中的微小粒子相互作用，并散射到不同的方向。DLS 分析通过测量散射光的强度和方向性来推断样品中的粒子 特性。

:::

## Q296. 杂质检测~蛋白聚体

:::answer

- **项目介绍：** 蛋白质聚集是蛋白类药物研发和商业化过程中一个重要的挑战之一。聚集体在蛋白质产品开发和生产过程的所有阶段都可以观察到，如蛋白表达、纯化、冻干等。蛋白质聚集可能对产品的质量和功效有显著的负面影响，其通常会减弱蛋白生物活性。此外蛋白聚集体可能存在潜在增强免疫原性的风险。 检测方法：SEC 法。SEC 测定技术基于蛋白质在固定相填充柱中的分子大小和构象的差异。固定相填充柱通常由多孔性材料构成，其中较大的分子无法进入孔隙，因此会相对较快地通过柱床，而较小的分子则会更多地进入孔隙中，导致其通过柱床的时间较长。

:::

## Q298. 杂质检测~HCP

:::answer

- **项目介绍：** 宿主细胞蛋白（HCP，HostCellProtein）是在生物制药（如生产重组蛋白药物、单克隆抗体）工艺过程中所产生的杂质，其超过一定含量则很有可能会引起免疫反应或其他不良反应。因此，HCP 残留量检测是抗体药物产品研发与质控中最常进行的分析项目，HCP 残留数值越低，则产品质控越好。宿主细胞蛋白（HCP）是由宿主细胞产生的与工艺相关的杂质，通常在重组生物制药产品中的含量较低，即使 HCP 杂质的总含量很低同样也可能对药物的性质造成不良影响。对于患者而言，HCP 是异源蛋白，通常大部分异源蛋白都可能具有免疫原性，因此即使很低浓度的 HCP 也可能引发不良免疫反应。宿主细胞的 HCP 中也包含一些蛋白水解活性酶，可使目的蛋白发生片段化，从而导致总产品降低。除了会导致目的蛋白降解和片段化以外，HCP 还会导致蛋白聚集等种种不良影响 检测方法：ELISA 是目前应用最广泛的 HCP 检测方法。该技术相对简单、精度良好，方便设定控制范围和建立技术规范。《中华人民共和国药典》2020 年版三部中的 HCP 检测均是要求采用酶

:::

## Q299. 杂质检测~HCD

:::answer

- **项目介绍：** 在重组蛋白类药物生产中，残余 DNA 作为细胞基质中潜在的危险因素，一直是人们关注的重点。残余 DNA 本身作为外源物质，可以引起机体的免疫反应。更为严重的是，重组蛋白质药物的细胞 DNA 残余可能引起细胞癌变的。有研究证实，代次在巧 0 代以下的 Vero 细胞不具致瘤性，代次超过 169 代即具有潜在的致瘤性。WHO 对于生产细胞已有规定，生产用 Vero 细胞的代次必须在 0 代以下。而药物生产中常用的 CHO 细胞，由于其调控生长的基因失调，具有无限寿命。一般认为其 DNA 具有使其他细胞生长失调和异变为癌细胞的潜能。因此，关于细胞 DNA 残余,WHO 推荐的标准是每剂量制品中的终 DNA 含量必须小于 10ng,美国 FDA 建议使用检测灵敏度达到 10pg 的检测方法检测制品中的 DNA 水平，《中国药典》三部（2010 版）规定每剂量疫苗中的 Vero 细胞 DNA 残留量必须少于 100pg 重组蛋白质药物的 DNA 残留量必须得到有效控制。 检测方法：在《中国药典》三部 2010 年版中，检测残余 DNA 的方法推荐为地高辛标记探针杂交法和荧光染色法。地高辛标记探针杂交法常见于膜斑点杂交法。该方法采用地高辛标记标准细胞基因组 DNA 片段，制成特异性探针，通过与待检品进行同源 DNA 杂交，检测制品中相应的细胞 DNA 残余。在膜斑点杂交法中，制成地高辛标记的探针是该方法的关键。

:::

## Q300. 杂质检测~蛋白 A

:::answer

- **项目介绍：** 残留 proteinA 因其可与免疫球蛋白高亲和力结合而被广泛用于 FC 融合蛋白或抗体类药物的纯化工艺中。但正是由于 ProteinA 与药物蛋白的结合力较强，在洗脱过程中需要使用较高强度的洗脱液使蛋白脱离，在这种洗脱环境下，经常会造成 proteinA 介质浸出。浸出的 ProteinA 作为外源蛋白，可能会诱导免疫应答，同时，作为金黄葡萄球菌的膜蛋白，其本身就具有促有丝分裂的作用。因此，监控 proteinA 残留不仅是药物安全性的重要保障，更是检测工艺稳定性的重要指标。 检测方法：proteinA 种类比较单一，并且其与抗体分子具有较强的结合力,ELISA 法自然成为检测 proteinA 残留的首选。目前国内多采用商品化的 ProteinA 检测试剂盒，操作方便，结果较精确。但在选用试剂盒检测 proteinA 残余时，部分 proteinA 仍结合着药物蛋白。由于位阻效应的影响，结合了药物蛋白的 proteinA 不能与试剂盒中的 proteinA 捕获抗体结合，同时，结合在 proteinA 上的药物蛋白会阻碍酶标抗体的结合，造成 ProteinA 残余的漏检。因此，选用商品化的 proteinA 检测试剂盒，尚需排除药物蛋白对检测的干扰，并进行干扰试验。

:::

## Q301. 活性检测~细胞增殖抑制实验

:::answer

- **项目介绍：** 生长因子类靶点（如 VEGF、HER2、EGFR）的抗体药物，通常通过阻断靶点和其受体的结合，从而抑制生长。因而在进行生物学活性测定时，可以采用细胞增殖抑制实验来测定。如贝伐珠单抗（VEGF 抗体）可以使用人脐静脉内皮细胞（HUVEC）增殖抑制法，曲妥珠单抗（HER2 单抗）可以使用表达 HER2 的乳腺癌细胞系进行增殖抑制实验，如 BT474 等。西妥昔单抗（EGFR 抗体）可以使用表达 EGFR 的肿瘤细胞系，如 A431 等。 检测方法：细胞增殖抑制实验。

:::

:::answer

纯度最好能达到 95%以上；否则每个循环有多个氨基酸出峰，无法归属到蛋白的序列。

## Q303. 肽段覆盖率检测方法

:::answer

目前，对蛋白质肽段覆盖率的检测根据药物申报要求的规定，均采用质谱法进行检测。 将样品通过酶切，进入质谱采集肽段的一级质谱信息和二级碎片离子质谱信息。采用软件对总离子流图进行分析，将测得肽段的相对分子量与理论值进行比较，并且通过二级质谱离子确认多肽的序列正确性。根据已确认肽段的氨基酸序列与理论序列的比值计算肽图覆盖率。

:::

## Q305. 除菌过滤器完整性检测

:::answer

除菌级过滤通常用于去除细菌等微生物，并确保最终产品在无菌方面发挥关键作用。终端除菌滤芯的完整性测试是一种必要手段。通过检测，可以确认滤芯是否安装正确，确认滤芯是否受损坏，确认滤芯是否符合制造规格，以确保在过滤器使用前后对过滤器性能进行验证。

## Q306. 翻译后修饰的检测分析

:::answer

肽图与液相色谱-质谱(LC-MS)相结合已成为药物开发过程中量化抗体 PTM 的基本分析技术。蛋白质 PTM 的识别可通过自上而下（Top-down）、中间向下（Middle-down）和自下而上（Bottom-up）的方法实现的。 其中 Bottom-up 分析方法是分析 PTM 最为成熟和最广泛使用的方法。该方法通过蛋白酶将蛋白切成较小的肽段，再通过串联质谱确定肽段的质量、它们各自的氨基酸组成以及 PTM 的位置。

:::

## Q307. 基因拷贝数的含义，检测方法，和丢失的原因？

:::answer

√01 基因拷贝数(copynumber)是指某一种基因或某一段特定的 DNA 序列在单倍体基因组中出现的数目。 √02 常用的检测方法为 Southernblot 法和荧光定量 PCR 法。 √03 丢失原因与目的基因插入到宿主细胞染色体中的稳定性、筛选压力、与整合位点、外源基因重组等等有关。

:::

## Q308. GMP 细胞库表征需要做那些检项？

:::answer

√01 中国申报应该按照中国药典《生物制品生产检定用动物细胞基质制备和检定规 程》进行；美国申报应按照 USP 规定进行。 √02 鉴定项目一般包括细胞鉴别、细菌/真菌检查、分枝杆菌检查、细胞内外源病毒因子检查、成瘤性和致瘤性检测等。

:::
