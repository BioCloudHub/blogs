## Q208. 08-C-端赖氨酸缺失

:::answer

抗体分子重链 C 末端的修饰包括 C 末端赖氨酸的切除以及脯氨酸的酰胺化，研究表明，抗体分子(gGl、1gG2 以及 lgG3)重链 C 末端都以脯氨酸甘氨酸赖氨酸(Pro-Gly-Lys)的氨基酸序列结尾，其中末端的赖氨酸会在细胞培养过程中被胞内外的碱性羧肽酶(Carboxypeptidases)切除，暴露出倒数第二个氨基酸甘氨酸，该过程被称为 C 末端赖氨酸的切除。但是，在很多情况下该赖氨酸残基不能被完全切除，导致得到的抗体中含有 C 末端赖氨酸未切除的结构变体，这部分变体被称为赖氨酸变体。由于赖氨酸是一个碱性氨基酸，它的不完全切除会引起蛋白质等 电点的上升。 研究表明，抗体分子 C 末端赖氨酸的切除对补体的激活具有十分重要的意义,Diebolder 等发现抗原抗体结合后，抗体之间需要通过非共价键形成六聚体，形成的六聚体将进一步招募和激

活补体蛋白 C1，进而发挥抗体药物的 CDC 活性。由于六聚体是由抗体的 C 末端之间通过非共价键形成的，因此抗体 C 末端的氨基酸序列以及电荷分布很有可能会影响到六聚体的形成。 虽然通过动物细胞培养表达的抗体中赖氨酸变体含量一般都能控制在 50%以内，但是由于抗体 C 末端的赖氨酸残基在抗体的合成、组装以及分泌等过程中所发挥的作用尚不清楚，同时为了保证产品质量的批间稳定性，生产过程中赖氨酸变体含量的检测与控制仍然显得十分重要。 < 单抗电荷变体||第三节电荷变体的形成机理一 >

## Q209. 什么是培养基模拟灌装？

:::answer

培养基灌装（Mediafill），是使用无菌培养基代替药液进行无菌生产流程的操作。培养基模拟灌装的过程中，使用微生物培养基代替药液测试无菌流程是否能够防止药品生产过程中的污 染。培养基模拟灌装是无菌生产工艺验证的一部分。 培养基灌装模拟的是整个工艺过程，包括混合和灌装。培养基灌装是用于评估整个无菌工艺过程，用以表明如果严格按照工艺要求生产，产品的无菌性有可靠保证。但必须承认，不管模拟试验还是实际生产总是存在差异，因此不能将培养基灌装结果与实际生产中的污染概率等同起 来，即使获得很好的培养基灌装结果，也不能排除实际生产发生污染的可能性。 培养基灌装应在与实际生产相同的条件（环境，设备等）下尽量模拟实际生产完成操作。应充分考虑生产时可能造成污染的各种因素，应对常规生产中的各种干扰活动进行模拟，应模拟常 规生产可能遇到的最差状况下的操作。

:::

## Q210. 培养基模拟罐装法律法规的要求

:::answer

- **中国 GMP 附录 1：** 无菌药品第十章生产管理第四十七条无菌生产工艺的验证应当包括培养基模拟灌装试验。 1）“应当根据产品的剂型、培养基的选择性、澄清度、浓度和灭菌的适用性选择培养基。应当尽可能模拟常规的无菌生产工艺，包括所有对无菌结果有影响的关键操作，及生产中可能出现的各 种干预和最差条件。” 2）“培养基模拟灌装试验的首次验证，每班次应当连续进行 3 次合格试验。空气净化系统、设备、生产工艺及人员重大变更后，应当重复进行培养基模拟灌装试验。培养基模拟灌装试验通常应当按照生产工艺每班次半年进行 1 次，每次至少一批。

:::

## Q211. 对培养基有什么选择的要求？

:::answer

如无特殊要求，所用培养基能支持的微生物生长菌谱范围要足够宽，应能促进革兰阳性菌、革兰阴性菌、酵母菌和霉菌的生长。通常可以选择大豆胰蛋白胨液体培养基。厌氧培养基（如硫乙醇酸盐液体培养基）应在特殊情况下使用。在培养结束后应进行微生物促生长挑战试验。促生长试验的指示菌除了相关药典规定以外，还应考虑使用环境监控和无菌检查中的分离菌，以代表实际生产中的可能遇到的污染菌。促菌生长试验的接种量应不超过 100CFU。如促生长试验失败，应进行调查并重复培养基灌装试验。 每瓶（单元）容器在培养前和培养中间阶段都要来回翻转，这样确保容器的内表面和密封件（如胶塞）内表面跟培养基完全接触。培养基应保持澄明，这样检查时能够很容易观察出由于污染导致的浑浊和沉淀现象。

:::

## Q212. 培养基促生长能力试验

:::answer

应考虑在无菌工艺模拟试验开始前及 14 天培养后按照现行中国药典方法对培养基进行促生长试 验。 根据所选择的培养基，按照中国药典要求进行促生长能力试验使用的菌种包括：白色念珠菌（CMCC98001）、黑曲霉（CMCC98003）、枯草芽孢杆菌（CMCC63501）、金黄色葡萄球菌（CMCC26003）、铜绿假单胞菌（CMCC10104）和生孢梭菌（CMCC64941）（必要时）等。除标准菌株之外，还可考虑加入环境监测和无菌检查中发现的典型微生物。 促生长试验接种量应小于 100CFU，按照中国药典要求培养，以证明培养基能够支持微生物的生长。

:::

## Q213. 细胞培养基中的成分介绍

:::answer

动物细胞最为早期的培养基来源于天然的成分，利用血浆、血清或者胚胎提取物进行培养。细胞能够达到在体外进行培养以后，人们希望能够找到成分更加确定的培养基，动物细胞培养基的发展历程，几乎满足这个想法，就是使用各种化学成分确定的物质代替化学成分不确定的天然成分。而现在所使用的动物培养基，根据添加物的类别可以分为：含血清培养基、无血清培养基、无蛋白培养基以及化学成分限定培养。动物细胞培养发展至今，有很多成熟的商业化培养基，但是针对各类商业化培养基，其配方基本上都是作为商业机密的，所以无法获取，我们只能从一些基础的培养基出发，去积累关于培养基的知识。 细胞培养基的基本组成：葡萄糖、氨基酸、无机盐、微量元素、维生素、缓冲液、生长因子、脂类、水。 氨基酸是 CHO 细胞培养基的关键成分，特别是在化学限定培养基中。研究表明，细胞培养基中氨基酸组成的微小变化可以改变生长曲线和产物滴度，也可以显著影响产物的糖基化模式。

:::

## Q214. 培养基中氨基酸概述

:::answer

氨基酸可以分为必须氨基酸(用于维持活细胞的氮元素平衡)和非必需氨基酸(可以由前体细胞合成)。必须氨基酸：组氨酸，异亮氨酸，亮氨酸，赖氨酸，甲硫氨酸，苯基丙氨酸，苏氨酸，色氨酸，缬氨酸，半胱氨酸，谷氨酰胺。非必须氨基酸：L-丙氨酸，L-精氨酸，天冬酰胺，天冬氨酸，谷氨酸，谷氨酰胺，甘氨酸，脯氨酸，丝氨酸，酪氨酸。它们都是细胞用以合成蛋白质的必需原料，不能由其他氨基酸或糖类转化合成。

:::

## Q216. 培养基中的非必需氨基酸

:::answer

非必需氨基酸包括丙氨酸、精氨酸、天冬酰胺、天冬氨酸、半胱氨酸、谷氨酸、谷氨酰胺、甘氨酸、脯氨酸、丝氨酸和酪氨酸。尽管哺乳动物细胞可以在培养中合成非必需氨基酸，但大多数细胞培养基仍然含有大部分或全部这些氨基酸，以支持细胞生长和产物生产。事实上，大多数非必需氨基酸对细胞培养过程有显著影响。

:::

## Q217. 丙氨酸

:::answer

丙氨酸的初始浓度相对较低。培养基研究表明，在细胞培养过程中，细胞要么消耗丙氨酸的速度很慢，要么从其它氨基酸中产生丙氨酸。培养基中有较高的乳酸时，丙氨酸产量与氨基酸浓度呈正相关。除非丙氨酸浓度超过 3mm，否则丙氨酸的积累对细胞生长几乎没有直接的抑制作用。 在乳酸和丙酮酸饥饿的条件下，丙氨酸倾向于转化为丙酮酸，并通过谷氨酸降解将游离氨释放到培养基中，从而增加了培养物中氨的浓度。

:::

## Q218. 精氨酸

:::answer

在抗体产生过程中，CHO 细胞以较高的速率消耗精氨酸。精氨酸缺乏可导致对数期的 CHO 细胞在 G0 期或 G1 期被捕获。培养基中过量的精氨酸可有效降低酸性产物的比例。 高浓度的精氨酸会抑制细胞生长，降低细胞活性。此外，精氨酸浓度的升高已被证明会使 c 端赖氨酸异构体的数量增加一倍，从而增加抗体产物的异质性。

:::

## Q219. 天冬酰胺和天冬氨酸

:::answer

天冬酰胺和天冬氨酸是细胞培养基中重要的非必需氨基酸，在关键代谢途径中起着核心作用。与其它氨基酸相比，CHO 细胞倾向于以相对较高的速率消耗天冬酰胺和天冬氨酸，特别是在对数生长阶段。就像葡萄糖、谷氨酰胺和谷氨酸一样，这两种氨基酸对 TCA 循环的补充和能量代谢都很 重要。 天冬氨酸和天冬酰胺通过转氨酶反应在谷氨酰胺和谷氨酸之间的转变中充当氨供体和受体，可生成谷氨酸和草酰乙酸。

:::

## Q221. 丝氨酸

:::answer

丝氨酸是 CHO 消耗第二高的氨基酸，也是 CHO 单碳单位代谢的主要供体。培养基中 CHO 细胞消耗的超过 70%的丝氨酸转化为甘氨酸，并最终参与胸腺嘧啶合成。尽管丝氨酸发挥着重要的作用，但培养基中低浓度的丝氨酸(1mM)除了在对数期降低甲酸盐和甘氨酸的产量，似乎对细胞生长没有什么影响。 然而，培养基中丝氨酸的耗竭会产生负面影响，包括增加天冬酰胺的消耗、丙氨酸的产生、乳酸的产生和氨的产生，最终会抑制细胞生长。考虑到高消耗率和耗竭的不利影响，通常在培养基中以相对高浓度提供丝氨酸。事实上，一些研究已经证明，在培养基中添加丝氨酸可以提高滴度和特异性峰细胞密度。

:::

## Q223. 培养基中的维生素介绍

:::answer

维生素是维持细胞生长的生物活性物质，在细胞代谢中起调节及控制作用。维生素可分为水溶性和脂溶性两类，水溶性维生素主要包括泛酸、维生素 B12、叶酸、烟酰胺、吡哆醛、硫胺素、核黄素、维生素 C、胆碱、肌醇等；脂溶性维生素主要包括维生素 A、D、E、K 等；有的培养液中还直接采用 ATP 和辅酶 A；大部分培养基中还有生物素。 许多维生素参与构成各种酶的活性基团的成分，没有它们，酶便没有活性，代谢活动将无法进行。比如，泛酸可以在细胞内转变成酰基载体蛋白和辅酶（如辅酶 A），参与糖类、脂类和蛋白质代谢中的催化反应；维生素 B12 则在细胞内参与叶酸的合成和脂肪酸的合成等。不同配方的细胞培养基中维生素的浓度有较大差异。例如，DMEM 培养基中维生素的含量约为 MEM 培养基的两倍，其原因是一方面不同种类维生素的作用不同，另一方面不同种类的细胞对维生素的需求也 可能有较大差异，相应细胞培养基的配方中维生素的含量也可能不同。

:::

## Q225. 叶酸(维生素 B9)

:::answer

- **介绍：** 也称为蝶酰单谷氨酸 13，由 2-氨基-4-羟基-6-甲基喋啶、对氨基苯甲酸和 L-谷氨酸三部分组成。叶酸存在多种还原形式和取代形式，在细胞内可以在二氢叶酸还原酶（抑制剂：氨基蝶呤）作用下叶酸、二氢叶酸和四氢叶酸进行相互转化。通过叶酸、5-甲基四氢叶酸或 5-甲酰基四氢叶酸的形式，以被动运输或者和 Na+耦联蛋白转运进入胞内，进入后单谷氨酰叶酸转化为聚谷氨酰叶酸形式，以防止向胞外运输。 稳定性：叶酸在加热、酸、光照和还原剂条件下会失活。叶酸在中性和酸性条件下溶解度低，需要碱溶，在酸调至中性 pH 形成过饱和溶液。过饱和溶液的稳定性在一定区间（取决于化合物的热力学和溶剂），在达到 pH 值饱和状态的边界时会发生沉淀。叶酸在 pH 值低于 6.0 时会发生沉淀或降解。

:::

## Q226. 氰钴胺素(维生素 B12)

:::answer

- **介绍：** 钴胺素在天然状态下（如胞内）不存在氰基，氰基是在从细胞中分离过程中获得。作为钴中心配位络合物，由 5，6-二甲基苯并咪唑核糖核苷酸通过氨基丙基与咕啉环（4 个吡咯基组成）连接形成，钴与 6 个氮配位，4 个是吡咯基的氮配位，1 个是二甲基苯并咪唑基的氮，另 1 个配位可以是-CN（氰钴胺素 21）、-CH3（甲基钴胺素）、-5’脱氧腺苷基（5'-脱氧-5'-腺苷钴胺素） 稳定性：在酸性或碱性条件下，钴胺素的侧链丙酰胺和乙酰胺水解成羧酸基团 26，产生无活性的降解产物。在碱性 pH 和升温条件下，B 环的乙酰胺侧链的环化形成内酰胺，产生无活性的脱氢维生素 B1227。在强酸性（H+）条件下，磷酸键的水解产生 cobinamide28 和苯并咪唑核苷酸 29，后者可进一步分解成多种产物。

:::

## Q227. 硫胺素(维生素 B1)

:::answer

- **介绍：** 硫胺素是通过亚甲基连接含硫的噻唑环和含氨基的嘧啶环组成，含有活性官能团-OH 和-NH2。转运至胞内后，-NH2 磷酸化形成功能性的辅酶形式硫胺素焦磷酸盐。硫胺素的磷酸盐形式是两性离子，在食品添加剂和培养基中是以阳离子形式存在，也有以阴离子形式的氯化物，后者相比较其它衍生物具有更高溶解度，但更常见的是硝酸盐形式。 稳定性：硫胺素对高温、氧化、还原和光敏感，降解机理和速率高度依赖于 pH 值。硫胺产生的降解产物有多种，溶液降解时通常会出现变色（由透明变为黄色）和沉淀现象，沉淀物质很少表征（如氧化产物脱氢硫胺素 31）

:::

## Q228. 维生素 B6

:::answer

- **介绍：** 维生素 B6 是吡啶衍生物，包含六个化合物：吡哆醇 48、吡哆醛 49、吡哆胺 50、以及三个的 5’-磷酸衍生物。六个化合物的化学性质不同，但具有相同的生理功能，且在胞内可以相互转化，胞内氧化还原活性酶会影响胺-醇-醛官能团之间的转化，磷酸激酶激活成磷酸盐形式。在培养基中至少需要包含上述的一种形式。在培养基中最常见使用的是盐酸吡哆醇，相对于其它形式，盐酸吡哆醇具有较好稳定性和较高溶解度，不易发生化学反应。 稳定性：维生素 B6 的六个形式对光都敏感，光解的主要产物是 4-吡哆酸 51。高温会引发吡哆醇二聚化形成 52 从而失去活性，长时间加热会导致二聚体继续二聚化。维生素 B6 的磷酸盐形式不稳定，容易发生水解。

:::

## Q229. 生物素（维生素 B7)

:::answer

- **介绍：** 生物素是由噻吩环和尿素形成的双环化合物，左侧链上连接一个分子的戊酸。生物素在各种酶促羧化反应中作为活化羧基载体。生物素本身直接作为羧化酶的辅酶，参与二氧化碳的固定，通过酰胺键和酶蛋白质赖氨酸残基的 ε-氨基连接，以生物素-赖氨酸（生物胞素）复合物形式存在。生物素在代谢过程中起关键作用，主要用于碳酸氢盐依赖性性的羧基化反应，如柠檬酸循环、脂肪酸合成、支链氨基酸的分解代谢。 稳定性：生物素理化性质稳定，在培养基暴露的条件下也不容易降解。在高温和紫外照射下，生物素在强酸或强碱条件下会灭活，其详细降解机理和降解产物尚未确定，唯一的已知降解产物来自过氧化氢或过氧化物自由基的氧化，过程中生物素的硫原子被氧化产生生物素亚砜 59 或生物素砜 60，生物素对氧气稳定。生物素会被抗生物素蛋白结合而失去活性。

:::
