## 什么是步骤二：细胞复苏，关键要点有哪些

:::answer

- **工序目的：** 水细胞复苏是从冻存的 WCB 或者 MCB 种子细胞水浴解冻，进行后续的传代扩增，为反应器生产提供种子细胞。 方法原理：水浴解冻。通常将细胞从液氮罐取出后在 37 度水浴锅中于 180 秒内进行解冻。 中间体控制：细胞复苏后的活细胞密度、细胞活率两个指标。《中国药典》三部规定：冻存前的细胞活力不低于 90%，复苏后的细胞活力不低于 80%

:::

## 什么是新建 WCB 前瞻性考虑，关键要点有哪些

:::answer

某产品是已上市单抗制品，在申报上市前已经建立了 MCB 和 WCB 两级细胞库，并获得了监管 部门的批准。现在 WCB 所剩不多，为了保障商业化生产的持续性，需要新建 WCB。

:::

## 什么是细胞库放行的基本条件，关键要点有哪些

:::answer

- **细胞库放行需要满足以下条件：** 1）按照 GMP 环境要求建立细胞库； 2）建库过程符合规定的操作规程； 3）建库物料的无动物源性证明； 4）细胞库检测和检定合格； 5）进行细胞活力的测试 < 生产用细胞库的制备、检定和管理 >

:::

## 什么是按照 GMP 环境要求建立细胞库，关键要点有哪些

:::answer

建库过程需要满足 GMP 要求。建库通常在 B 级或者 C 级背景下的生物安全柜进行无菌操作，传递窗应避免直接通向 C 级或者 CNC 区域 < 生产用细胞库的制备、检定和管理 >

:::

:::answer

通常外源病毒因子检测时间较久，3-6 个月，是细胞库放行的限速点。 < MCB 和 WCB 细胞库检定项目技术指导 >
