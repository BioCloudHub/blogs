## Q221. HCP 检测覆盖率研究

:::answer

## Q223. 残留物的去除及限度要求

:::answer

生产用原材料在生物制品中的残留物可能因其直接的毒性反应、外源因子污染或有害的免疫应答，引发受者产生不良反应或影响产品效力。生产过程中应尽可能采用经去除和（或）灭活外源因子的生物原材料，或采取相应措施对这些原材料中可能存在的外源因子、致病物质或与该材料相关的特定污染物予以去除和（或）灭活，去除和（或）灭活工艺应进行验证。 应通过验证结果评价生产工艺对已知毒性原材料去除的一致性，或采用批放行检测，以证实所去除的毒性原材料已达到安全水平，残留有机溶剂应符合本版药典“残留溶剂测定法”的相关要求。

:::

## Q224. 物料的放行一般进行什么检测项目？

:::answer

- **物料放行通常进行：** 性状、鉴别、含量、微生物限度、内毒素等检测项目。

:::

## Q225. 培养基放行一般进行什么检测项目？

:::answer

- **培养基放行通常进行：** pH、渗透压、浊度、微生物限度、内毒素等检测项目。

:::

## Q226. 参比品建立的意义

:::answer

- **药物参比品：** 药物参比品是已获得批准的原始药物，通常由国家药品监管机构或国际药典委员会认定和提供。药物参比品用于确认药物的质量和疗效，并与新药物进行比较，以确保其与参考品的相似性。 使用：原液放行、稳定性研究、工艺变更、产品表征研究等

:::

:::answer

- **根据药物开发阶段和用途分：** 过渡期参比品、一级参比品、工作参比品。

## Q230. 工作参比品

:::answer

工作参比品应选择能代表商业化生产工艺的批次进行制备，主要用于日常的批放行检验、稳定性测试、中间体控制、表征测试等。工作参比品需要采用一级参比品进行标定。

:::

## Q231. 参比品标定的意义

:::answer

- **药物参比品：** 药物参比品是已获得批准的原始药物，通常由国家药品监管机构或国际药典委员会认定和提供。药物参比品用于确认药物的质量和疗效，并与新药物进行比较，以确保其与参考品的相似性。 参比品标定是通过比较参比品和待测样品的性质和成分来确定待测样品的质量。参比品标定的好处是显而易见的。它可以确保产品的质量稳定性和一致性,避免因质量问题而导致的安全事故和经济损失

:::

## Q232. 参比品标定内容

:::answer

- **对于单克隆抗体参比品，其标定项目为：** 含量和效价。

:::

## Q233. 参比品蛋白含量标定

:::answer

部分企业将检验确认蛋白含量的活动成为参比品蛋白含量标定，由于单抗蛋白含量通常采用紫外-分光光度法测定，无需采用参比品/标准品进行标定，称为参比品蛋白含量的测定更加合适。

:::

## Q234. 参比品效价的标定

:::answer

通常，单克隆抗体的参比品的标定通常是指效价的标定。 一级参比品作为活性参比的基准，通常具备确证临床批次代表性，并且进行了充分的结构确认，所以第一批一级参比品活性无需进行标定，直接赋值为 100%。也推荐使用过渡期参比品对一级参比品进行标定。 工作参比品采用一级参比品进行标定。

:::

## Q235. 参比品标定的要求

:::answer

第、效价标定检验方法应该经过验证。 第二、关注样品的代表性，在设计标定实验时，应考虑在罐装前种后随时取样。 第三、对工作参比品生物学活性进行赋值时，通常情况下若多批次测定相对效价的 95%置信区间位于原批次参比品的 90%~100%时，可将该新批次参比品赋值 100%。

:::

## Q236. 参比品确证内容

:::answer

- **参比品确证分两部分内容：** 参比品质量标准、参比品结构确证。 抗体药参比品分类基础知识

:::

## Q237. 参比品质量标准

:::answer

- **单克隆抗体参比品质量标准通常包含：** 鉴别、含量、一般理化(通常为 pH 和渗透压)、纯度（通 常为大小和电荷变异体）、效价（通常包含生物学活性和结合活性）。

:::

## Q238. 参比品结构确证

:::answer

参比品结构确证时对参比品的一级结构、高级结构、糖基化、Fc 效应功能进行确认。

:::

## Q239. 参比品确证的基本要求

:::answer

工作参比品应采用一级参比品作为基准进行确证，第一批一级参比品可采用过渡期参比品进行确证，后续一级参比品采用前一批一级参比品为基准进行确证。

:::

## Q240. 参比品确证数据引用

:::answer

当制备的级参比品和工作参比品来源于同批次制备时，工作参比品的确证可引用相关 的数据。

:::

## Q241. 参比品放行的意义

:::answer

为保证参比品质量和批间一致性做质量控制要求。 抗体药参比品分类基础知识

:::

## Q242. 参比品放行审核的内容

:::answer

- **单克隆抗体参比品放行审核内容通常包含：** 制备过程应符合规程或者方案的要求、已按照质量标准及结构确证方案完成确证，且结构符合规定；已完成标定；制备、检验、结构确证、标定等活动均有完整的记录。

:::
