# 生物分析

作为 CMC 分析开发与质量研究的支撑专题，聚焦组学和高级表征技术。

## 在 CMC 中的作用

- 支撑细胞株筛选与稳定性评估
- 辅助产物结构表征与质量属性研究
- 用于工艺机理分析和异常波动溯源

## 技术类别

### 基因组学
- 高通量测序分析
- 变异检测与注释
- 序列比对

### 蛋白质组学
- 质谱数据分析
- 差异定量分析
- 生物标记物发现

### 单细胞组学
- scRNA-seq 分析
- 细胞类型注释
- 轨迹推断
