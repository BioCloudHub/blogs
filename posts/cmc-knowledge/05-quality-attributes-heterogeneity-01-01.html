什么是 CHO 细胞表达蛋白 N 糖型的形成过程是怎样的，关键要点有哪些

回答BioCloudHub

简答： 主要在高尔基体和内质网中进行

  • 合成场所： 高尔基体和内质网是单克隆抗体糖基化合成的两个重要场所，在抗体糖基化合成的过程中，会有很多的前体与酶蛋白的参与。 在内质网表面上，GDP-Man 和 UDP-GlcNAc 在多聚醇-磷酸合成两个 N-乙酰葡萄糖胺和五个甘露糖的多糖复合物（GlcNAc2Man5）。 在内质网内，GlcNAc2Man5 添加额 3 个葡萄糖残基和 4 个甘露糖残基（3Glc-GlcNAc2Man9）。 3Glc-GlcNAc2Man9 连接到抗体 Asn297 上，在许多酶蛋白的作用下，切除 3 个葡萄糖残基和 1 个甘露糖残基。 携带 GlcNAc2Man8 的抗体进入高尔基体内，切除高甘露糖残基并添加一些寡糖（岩藻糖、唾液酸、半乳糖、乙酰葡萄糖胺），最后合成 N-糖结构。

糖型异质体形成的原因

回答BioCloudHub

GlcNAc（N-乙酰葡萄糖胺）和高甘露糖组成了 N-糖的核心，在此基础上添加唾液酸、岩藻糖、半乳糖等，通常为双天线结构。 N 糖型分子量大约占整个单抗的 1%-3%，在抗体药物的合成过程中，一般会伴随少量的唾液酸、岩藻糖、半乳糖的缺失，以及甘露糖的不完全切除形成高甘露的单抗，这些是糖型异质体形成的主要原因。

糖型异质体的调控有哪些方法

回答BioCloudHub

  • 选择合适的宿主细胞，不同的宿主细胞对蛋白质翻译后修饰能力有差异。 调控 pH，pH 对糖型修饰的影响的主要原因是胞外的 pH 会改变高尔基体的 pH 进而影响糖基化相关酶的酶活性，pH 值对半乳糖糖型的影响依据培养细胞和培养条件等的不同而表现出差异。 溶氧，溶氧主要影响半乳糖残基的水平，其影响机制是，一方面，低溶氧条件下半乳糖的前体物质尿苷二磷酸半乳糖（UDP-Gal）下降
  • 另一方面，溶氧并且低溶氧条件下氧化还原电位影响抗体二硫键的形成，而二硫键得到正在加入寡糖链的半乳糖产生的空间位阻，从而降低半乳糖残基的修饰。 培养基和添加剂，一是构成糖型的前体物质，如葡糖糖、半乳糖、甘露糖、氨基葡萄糖、乙酰葡萄糖胺等
  • 二是可以影响前体物质供应，如钙、镁、锰、铜等。 调控培养液的渗透压和培养温度。

常用添加剂对糖型的影响机制

回答BioCloudHub

葡萄糖的浓度对细胞合成糖型产生较大的影响，葡萄糖浓度偏低时（0-25 mm），葡萄糖将被细胞主要用来当做营养物质维持生存，而不被当做糖基化的前体物质，因而会导致产品的整体糖基化水平降低，并且非正常的糖基化结构产生增多。

添加 D-半乳糖可以提高胞内的 UDP-Gal 含量，降低 UDP-Glc，从而提高半乳糖糖基化修饰。

氨基葡萄糖（GlcN）或 N-乙酰葡萄糖胺（GlcNAc）可以提高胞内的 UDP-GlcNAc，从而提高甘露糖糖基化修饰，并降低糖型的复杂程度。

几夫碱的添加也有利于高甘露糖的增加。

添加乙酰甘露糖胺或者半乳糖、尿苷、Mn2+、地塞米松可以提高蛋白产品末端唾液酸糖基化修饰。

Mn2+可使非岩藻糖修饰增加。

氨是糖基化调节的重要物质，氨水平的增加可以造成胞内高尔基体 pH 增高，从而影响 UDP-GlcNAc 和唾液酸的转运。

温度对糖型调控有何影响

回答BioCloudHub

温度对单抗糖型的影响主要是由温度影响了糖型形成的关键酶所导致的。

低温的培养工艺会显著减少唾液酸及半乳糖含量。

在不同初始谷氨酰胺浓度条件下，降温导致半乳糖修饰降低，唾液酸修饰提升。

降温会导致细胞代谢状态的变化，减少细胞内核苷糖体供体数量，从而影响抗体 Fc 段糖基化水平。

在研发阶段，由于温度对细胞表达的蛋白产量与质量的影响是多方面的，在选择培养工艺时应该综合考虑。

常见的DO 对糖型的调控作用有哪些，各自怎么理解

回答BioCloudHub

  • 溶氧主要影响半乳糖残基的水平，其影响机制是，一方面，低溶氧条件下半乳糖的前体物质尿苷二磷酸半乳糖（UDP-Gal）下降
  • 另一方面，溶氧并且低溶氧条件下氧化还原电位影响抗体二硫键的形成，而二硫键对此正在加入寡糖链的半乳糖产生了空间位阻，从而降低半乳糖残基的修饰。 研究发现，DO 在较大的波动范围内（40%~100%），CHO 细胞的分泌蛋白的电荷异质体基本一致，且 G0、G1 水平没有明显的变化。因此，在细胞培养工艺的优化与开发过程中，应该给予综合考虑。

常见的渗透压对糖型异质体的调控作用有哪些，各自怎么理解

回答BioCloudHub

细胞培养基的基础培养基的渗透压一般在 280~320 mOsm/kg 之间，但随着补料的加入，培养液的渗透压会越来越高。渗透压对糖型的影响表现在多个方面。

  • 有研究报道，当在 CHO 细胞培养中加入氯化钠时： 随着渗透压的升高单抗的高甘露糖糖基化修饰（Man5）逐渐增高，G0F 的含量则随着渗透压的升高而降低，G1F 差异不大。 渗透压对 Man5 水平的影响可能是高渗透压条件导致 GnT-I 酶活性降低、mRNA 翻译减少导致的。

抗体电荷变体形成的原因是什么

回答BioCloudHub

抗体电荷异质性是由于抗体分子中翻译后修饰以及降解的发生导致其等电点(pI)或空间电荷分布等表面电荷特性发生了改变，进而导致电荷变体的生成。

抗体电荷变体的来源众多，生成机制十分复杂。

由于抗体在生产、存储以及使用过程中往往伴随着多种翻译后修饰和降解的同时进行，共同影响着抗体的表面电荷特性，因此抗体的电荷异质性是这些翻译后修饰和降解的综合效应。

抗体的电荷变体主要来自于生产过程中的一系列翻译后修饰和降解，包括自发和非自发的降解。

酸性电荷变体形成的原因是什么

回答BioCloudHub

天冬酰胺的去酰胺化(Deamidation) 赖氨酸的糖化(Glycation) 三硫键的形成(Trisulfide) 糖基化中的唾液酸(Sialic acid)

碱性电荷变体形成的原因是什么

回答BioCloudHub

C 末端赖氨酸的切除(Lysine clipping) N 末端焦谷氨酸的形成(pyroGlu) 天冬氨酸的异构化(Isomerization) 脯氨酸的酰胺化

电荷变体对药物研发有什么影响

回答BioCloudHub

  • 抗体电荷变体最终影响药物在临床使用时的有效性和安全性，是一类重要的质量属性。 影响抗体药物的结合能力(Binding)
  • 影响药代动力学(PK)
  • 影响生物学活性(Bioactivity)
  • 影响免疫原性(Immunogenicity)
  • 影响结构稳定性(Stability)。

常见的电荷变体的调控策略有哪些，各自怎么理解

回答BioCloudHub

温度是动物细胞培养过程中最为重要的过程控制参数之一。

一般哺乳动物细胞的培养温度在 37°C 左右，过低或过高的培养温度都有可能影响到细胞的生长、代谢、产物的表达以及质量，包括抗体的电荷异质性。

多项研究结果表明，降低培养温度可以有效降低单抗中酸性电荷变体的含量，同时提高主峰含量。

pH 是动物细胞培养过程中另一个重要的过程控制参数，它反映了培养体系的酸碱程度。

pH 的控制策略同样会影响到动物细胞的生长、代谢、产物的表达以及质量。

在电荷异质性方面，培养过程中的 pH 值主要会影响抗体的酸性电荷变体含量，pH 是影响单抗电荷分布的关键过程参数之一，降低培养体系的 pH 值可以有效降低单抗中酸性电荷变体的含量。

光照同样会引起抗体电荷变体的生成。

基因泰克公司研究者在进行单抗生产规模放大的过程中发现，小规模(2L)玻璃反应器中生产得到的单抗，其酸性电荷变体往往会高于大规模(400L)反应器。

冷模实验表明光照是引起小规模反应器中酸性电荷变体偏高的主要原因。

对酸性变体进一步分析后猜测光照可能会引起单抗空间结构以及空间电荷分布的改变，进而影响单抗的电荷异质性。

金属离子是动物细胞无血清培养基中一类重要的添加剂，参与着细胞内包括 DNA 和蛋白合成、有丝分裂、能量代谢、信号转导等在内的 300 多个重要的生物过程。

在抗体的电荷异质性方面，金属离子主要通过调节相关酶的酶活，进而影响电荷变体的生成过程。

研究指出，培养基中的铜离子和锌离子可以分别抑制和激活胞内外的碱性羧肽酶，影响抗体重链 C 末端赖氨酸的切除，从而影响最终产品的赖氨酸变体含量。

培养基中的许多添加剂，包括丁酸钠、地塞米松、氢化可的松等，同样影响着抗体分子的电荷异质性。

其中丁酸钠具有抑制组蛋白去酰基酶活性的功能，常常被添加到培养基中用来促进产物的表达。

近年来越来越多的研究结果表明，丁酸钠在调节抗体的糖基化、电荷异质性等质量方面同样发挥着重要的作用。

常见的去酰胺化/异构化形成的原因有哪些，各自怎么理解

回答BioCloudHub

天冬酰胺(Asn)的去酰胺化是抗体在生产以及存储过程中自发进行的一类重要的降解反应，它的存在会导致酸性电荷变体的形成。

在中性或碱性 pH 条件下，Asn 的去酰胺化过程通过形成一个五基团环状的中间体琥珀酰亚胺而进行。

首先，肽键上的氮原子发生去质子化作用，经去质子化后的氮原子随后攻击侧链上的羰基碳，丢掉一个氨基形成琥珀酰亚胺，该过程是一个不可逆反应。

形成的琥珀酰亚胺不稳定，经水解后形成天冬氨酸(Asp)和异构的天冬氨酸时该过程是一个可逆反应。

因此，抗体分子中的 Asp 位点也可以自发脱水降解形成琥珀酰亚胺，同时形成的琥珀酰亚胺可以进一步水解形成 Asp 和 isoAsp，该过程被称为 Asp 的异构化。

在酸性 pH 条件下，Asn 的去酰胺化过程则主要通过直接水解侧链上的氨基而进行，同时仅生成 Asp。

由于 Asn 的等电点高于三种可能的产物，即 Succinimide、Asp 以及 isoAsp，因此 Asn 的去酰胺化会导致抗体酸性电荷变体的形成，而 Asp 的异构化则会导致抗体碱性变体的形成。

常见的去酰胺化/异构化的调控策略有哪些，各自怎么理解

回答BioCloudHub

Asn 去酰胺化以及 Asp 异构化过程的速率受氨基酸序列、空间构像等蛋白结构的影响。

研究表明，当 Asn 以及 Asp 碳末端的氨基酸为甘氨酸(Glycine，Gly)或丝氨酸(Serine，Ser)时，去酰胺化以及异构化的速率将大大提高。

因此，在进行抗体分子设计时，需要避免这样的氨基酸序列出现，以免影响药物的结构和功能。

Asn 去酰胺化以及 Asp 异构化过程的速率还受环境温度、pH 值、离子强度等因素的影响。

Pace 等通过冷模实验研究了不同缓冲液体系、pH 值以及温度对抗体 Asn 去酰胺化过程的影响，结果表明，随着温度的升高，去酰胺化速率显著提高，同时随着 pH 的下降，去酰胺化速率呈现出先下降后上升的趋势。

回答BioCloudHub

  • 异构化的发生则导致单抗的药效下降了 80-90%。为了提高 Herceptin 的稳定性，Yang 等将 Asn30 以及 Asp102 两个氨基酸分别替换成了 Gn30 以及 Gu102，结果显示，抗体的稳定性得到了大幅度的提高，同时抗体的结合活性并未受到影响。因此，Asn 去酰胺化以及 Asp 异构化往往作为抗体药物的 CQA 进行考量。

常见的C 末端修饰形成的原因有哪些，各自怎么理解

回答BioCloudHub

抗体分子重链 C 末端的修饰包括 C 末端赖氨酸的切除以及脯氨酸的酰胺化。

研究表明，抗体分子（IgG1、IgG2 以及 IgG3）重链 C 末端都以脯氨酸-甘氨酸-赖氨酸（Pro-Gly-Lys）的氨基酸序列结尾，其中末端的赖氨酸会在细胞培养过程中被胞内外的碱性羧肽酶（Carboxypeptidases）切除，暴露出倒数第二个氨基酸甘氨酸，该过程被称为 C 末端赖氨酸的切除。

在很多情况下，该赖氨酸残基不能被完全切除，导致得到的抗体中含有 C 末端赖氨酸未切除的结构变体，这部分变体被称为赖氨酸变体。

由于赖氨酸是一个碱性氨基酸，它的不完全切除会引起蛋白质等电点的上升，因此赖氨酸变体是一类碱性电荷变体。

由于抗体具有两条完全相同的重链，因此在进行 C 检测时能观察到两个由于赖氨酸变体而形成的碱性峰 K1 和 K2，其中 K1 是由于单条重链上的赖氨酸未被切除而形成的碱性峰，而 K2 则是由于两条重链上的赖氨酸均未被切除而形成的碱性峰。

此外，当 C 末端赖氨酸被切除后，暴露出来的甘氨酸以及脯氨酸又会在另一种酶系，即肽酰甘氨酸 N-酰胺化酶（PAM）的催化下发生甘氨酸的切除和脯氨酸的酰胺化。

由于脯氨酸的酰胺化使抗体分子额外增加了一个碱性基团（氨基），因此同样会导致碱性电荷变体的生成。

C 末端修饰对药物有何影响

回答BioCloudHub

研究表明，抗体分子 C 末端赖氨酸的切除对补体的激活具有十分重要的意义，研究发现抗原抗体结合后，抗体之间需要通过非共价键形成六聚体，形成的六聚体将进一步招募和激活补体蛋白 C1，从而发挥抗体药物的 CDC 活性。

由于六聚体是由抗体的 C 末端之间通过非共价键形成的，因此抗体 C 末端的氨基酸序列以及电荷分布很有可能会影响到六聚体的形成。

虽然通过动物细胞培养表达的抗体中赖氨酸变体含量一般都能控制在 50%以内，但是由于抗体 C 末端的赖氨酸残基在抗体的合成、组装以及分泌等过程中所发挥的作用尚不清楚，同时为了保证产品质量的批间稳定性，生产过程中赖氨酸变体含量的检测与控制仍然显得十分重要。

抗体聚体形成的原因是什么

回答BioCloudHub

  • 聚体（Aggregate）通常由抗体分子间通过共价键或氢键等分子间作用力形成的聚合物，是由两个或多个蛋白组成的稳定复合物，其形成过程十分复杂，包括：蛋白结构破坏，形成聚合物
  • 蛋白空间构象的改变，形成不可逆的聚合物。

两个蛋白结合在一起形成二聚体，在此基础上添加单体，使得聚体变大。

单抗的 Fab 段和 Fc 段的错误折叠、未折叠及正确折叠均可形成聚体，并且当可溶聚体分裂后便不再溶解。

蛋白质中二硫键也是聚体形成的因素，半胱氨酸上未进行结合的自由硫醇基会导致二硫键含量的减少，进而蛋白发生错误折叠。

抗体聚体有哪些调控手段

回答BioCloudHub

在细胞培养过程中可以通过调节培养液的理化环境来控制抗体的聚体水平，如温度、pH 以及渗透压等。

  • Cromwell 等人研究发现降低培养环境的温度可以降低聚体水平
  • 聚体水平还会随着培养液 pH 和渗透压的升高而降低，增加培养液中氯化钠的含量可以提高渗透压，进而降低聚体水平
  • 此外，一些培养基关键组分，如谷胱甘肽、半胱氨酸、胱氨酸和硫酸铜等氧化还原物质可以减少 CHO 细胞培养过程中抗体聚体的形成
  • 改变蛋白电荷的分布，如通过突变暴露的氨基酸残基、在抗体末端增加带电多肽
  • 研究指出带负电荷的氨基酸可加强蛋白间的静电排斥作用力，从而能够抑制聚体的形成
  • 蛋氨酸残基氧化对多种蛋白形成聚体的倾向性具有重要的影响，为防止蛋氨酸残基的氧化，冷冻通常是抗体储存及运输中的选择，从而可减少聚体的形成
  • 研究发现，培养周期越短，聚体的含量就越低，因此，当产量达标时，上游可通过减少培养时间来降低聚体含量
  • 对 pH 和离子强度的高低可影响抗体所带电荷，增加蛋白的静电排斥作用力，从而抑制聚体的形成
  • 通过基因手段可以阻碍聚体的形成，如突变暴露的氨基酸残基、在抗体末端增加带电多肽、促进 CDRs 边缘氨基酸突变等
  • 对抗体生产、储存环境的改变也可增加聚体形成阻力，如 pH、溶液离子强度、温度等。这些措施均可增加单抗的表面电荷，从而增强单体与单体间的静电排斥作用，用以抵消“hotspots”间的强疏水作用力，抑制聚体的形成。

下游可采用哪些方法降低聚体水平

回答BioCloudHub

如果细胞培养过程中不能去除高聚体含量，则可通过纯化工艺的优化来降低聚体的含量：

  • 纯化中缓冲液的类型是影响聚体含量的重要因素，在纯化中，Protein A 洗脱抗体时使用的酸性缓冲液会导致聚体的形成
  • 研究发现，与柠檬酸盐、甘氨酸及组氨酸相比，此步骤的缓冲液中精氨酸的加入可降低收获液中聚体的含量
  • SEC 柱析可根据单体与聚体的大小不同，而将聚体去除，但会牺牲部分回收率
  • HIC 和超滤也可用于聚体的去除
  • AEX 和 CEX 降低聚体率的机理主要在于其可将二聚体或多聚体分解成单体，最优条件下的聚体含量可降至 0.5%以下。

抗体片段的形成对药物有什么影响

回答BioCloudHub

抗体还原型片段的存在对抗体药物的安全性和有效性具有多种潜在的负面影响，包括降低药物的生物学活性、缩短药物在体内的半衰期、引起人体的免疫反应、对病人的安全存在潜在的威胁等。

抗体聚体的形成对药物有什么影响

回答BioCloudHub

单抗聚体含量高，纯化收率低，成本相对较高。

在治疗过程中，聚体的存在不仅会降低药物的效果，还会引起机体的免疫反应，具有潜在的副作用。

在动物免疫原性研究的实验中发现，聚体中天然结构比例以及聚体的大小决定其免疫原性的强弱，因此有学者推断，聚体的免疫原性是由其表面的抗原表位的重复性决定的。

聚体的形成过程是怎么样的

回答BioCloudHub

  • 蛋白天然结构的破坏，形成可逆自聚合。 由于构象的改变，使这种可逆的自聚现象变得不可逆转。 其他单体加入使聚体变大，聚体与聚体相互作用形成可溶性聚体团，聚体大小再次增加
  • 可溶性聚体团相互分离。 由于构象的改变，使这种可逆的自聚现象变得不可逆转，导致不溶性聚体产生。
