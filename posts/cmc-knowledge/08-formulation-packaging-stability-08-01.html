什么是冻干

回答BioCloudHub

  • 冻干是一种以从液体配方中去除溶剂为核心原理的工艺过程,是目前制造和储存抗体等蛋白质类药物的重要手段．从配方中除去或抑制溶中促进化学和物理降解途径的各种影响因素,可以显著延长药物制剂的保质期．尽管冻干价格昂贵,但仍然是保持这些高活性分子稳定性的一种 非温和的制剂方法．冻干工艺涉及两个重要的过程: 冷冻,在此过程中大部分溶剂转化为冷冻固体
  • 干燥,在此期间几乎所有溶剂(冷冻和未冷冻)从配方中除去．基于干燥的机理,将干燥过程进一步分为两个步骤,即初级干燥和二次干燥。 < Q1891~1900||生物药制剂(DP)稳定性研究总结 >

什么是冻干制剂的优缺点，关键要点有哪些

回答BioCloudHub

  • 优点： 保存时间长：冻干制剂工艺可以去除大部分水分和有机溶剂，从而大大延长了产品的保存时间。 稳定性高：通过冻干处理，可以使产品在常温下稳定存在，不易发生化学反应和生物降解。工艺过程对组分的破坏程度最小,热畸变极其微弱，对不耐热药物特别适合。 易运输：由于水分和有机溶剂的去除，冻干制剂的体积和质量大大减小，使得产品的运输变得 更加方便。 可重复使用：冻干制剂工艺可以使产品在需要时进行复溶，因此可以重复使用。 应用广泛：冻干制剂工艺可以应用于各种药物、酶、抗体等生物制品的保存和运输。制品为多孔结构，质地疏松，较脆，复水性能好，重复再溶解迅速完全，便于临床使用。 缺点： 成本高：冻干制剂工艺需要使用大量的冷冻剂和干燥剂，同时还需要进行长时间的低温处理， 因此生产成本较高。 工艺时间长（典型的干燥过程周期需要二十小时左右），生产成本高 对成分有影响：在冻干过程中，由于需要低温处理，可能会对某些成分的稳定性产生影响。 需要特殊储存条件：由于冻干制剂需要低温储存，因此需要特殊的储存条件，这可能会增加产品的管理和运输成本。 可能产生玻璃化转变：在冻干过程中，产品可能会经历玻璃化转变，这可能会影响产品的性质和稳定性。

什么是冷冻干燥原理，关键要点有哪些

回答BioCloudHub

冷冻干燥是利用冰晶升华的原理，在高度真空的环境下，将已冻结了的食品物料的水分不经过冰的融化直接从冰固体升华为蒸汽，一般真空干燥物料中的水分是在液态下转化为汽态而将食品干制，故冷冻干燥又称为冷冻升华干燥。

水有三相，O 点为三相共点，OA 为冰的融解点。

根据压力减小、沸点下降的原理，只要压力在三相点压力之下（压力为 646.5Pa 以下，温度 0℃ 以下），物料中的水分则可从水不经过液相而直接升华为水汽。

根据这个原理，就可以先将食品的湿原料冻结至冰点之下，使原料中的水分变为固态冰，然后在适当的真空环境下，将冰直接转化为蒸汽而除去，再用真空系统中的水汽凝结器将水蒸汽冷凝，从而使物料得到干燥。

这种利用真空冷冻获得干燥的方法，是水的物态变化和

移动的过程，这个过程发生在低温低压下，因此，冷冻干燥的基本原理是在低温低压下传热传质的机理。

冷冻干燥不同于普通的加热干燥，物料中的水分基本上在 0℃ 以下的冰冻的固体表面升华而进行干燥，物质本身则剩留在冻结时的冰架子中，因此，干燥后的产品体积不变、疏松多孔。

冰在升华时需要热量，必须对物料进行适当加热，并使加热板与物料升华表面形成一定温度梯度，以利于传热的顺利进行。

什么是冻干保护剂

回答BioCloudHub

  • 冻干过程会影响蛋白质的稳定性，通常需要添加冻干保护剂，常见的冻干保护剂如下： 按照分子量分类可以分为：低分子化合物和高分子化合物。一般认为，低分子化合物在冻干过程中直接发挥作用，而高分子化合物则是促进低分子化合物的保护作用。因此，制备保护剂配方 时，一般将低、高分子化合物配合使用。 按照冻干保护剂功能和性质分类可以分为：冻干保护剂、填充剂、抗氧化剂、酸碱调整剂。冻干保护剂，在冻结和干燥过程中，可以防止活性组分发生变性。填充剂，能防止有效组分随水蒸汽一起升华逸散，并使有效组分成形。抗氧化剂，用作防止生物制品在冷冻干操过程以及储藏过程中发生氧化变质。酸碱调整剂，在冷冻干澡过程和储藏过程中，能将生物制品的 pH 调整到活性物质的最稳定区域。 按照物质的种类分类可以分为：糖、醇类、氨基酸类、盐类等。糖类，是冻干品中最常用的赋形剂，其结构中的羟基，可以取代蛋白质和水之间的的氢键来提供稳定的效果。醇类，像甘露醇一般用作填充剂，其在慢速冻结时会结晶，从而为活性组分提供支撑结构，且不与活性组分发生反应。甘油一般用作低温保护剂。糖和醇类对生物制品冷冻干燥过程中的保护与其添加的浓度直接相关，当浓度达到一定的程度之后，对蛋白的稳定作用能力达到极限，过高浓度反过来会使得冻干过程中蛋白质发生变性。氨基酸离子具有酸碱两性，因此能够在生物制品的低温保存和冷冻干燥过程中抑制溶液的 pH 变化，从而达到保护活性组分的目的。此外许多氨基酸还是很好的填充 剂，例如甘氨酸。盐类，主要用作缓冲剂，平衡冷冻干燥保存过程中的 pH 值。

什么是冻干制剂处方开发一般考虑，关键要点有哪些

回答BioCloudHub

冻干制剂处方开发一般考虑 API 性质、冻干溶剂、冻干保护剂、其他附加剂。

API 性质。

API 的理化性质决定着后续辅料及冻干工艺的选择。

API 的溶解度，拟配制的浓度，装量，最高耐受的温度，化学稳定性，与辅料、包材的相容性等这些都是最初需要考虑的方面。

冻干溶剂。

常用溶剂包括注射用水、乙醇、丙二醇、正丁醇、异丙醇、叔丁醇、聚乙二醇。

一般来说，注射剂溶剂首选注射用水，当注射用水不能满足药物性质和临床要求时，才会考虑其他非水溶剂。

理想的冻干非水溶剂需要如下条件：凝固点高、挥发性强、能与水混溶。

什么是升华干燥阶段制品的塌陷，关键要点有哪些

回答BioCloudHub

  • 冻干产品在升华干燥阶段，随着升华时间的增长，产品出现已干层和冻结层，两层之间的交界面是升华面
  • 升华过程中，升华面不断减小冻结层，不断增加已干层厚度。已干层结构疏松多孔，以便冻结层的水蒸气能逸出。但某些已干燥的产品当温度升高时，可能会失去刚性，变得有粘性，发生类似坍塌的现象。失去多孔结构，会堵塞升华时水蒸气的逸出通道
  • 使升华 速率变慢，搁板供给冻结层的热量有剩余，导致冻结层产品温度上升，当温度达到共晶点温度以上时，产品会发生熔化或发泡现象，此时的温度叫塌陷温度。 有些产品崩解温度高于共晶温度，升华时只需控制产品温度稍低于共晶温度即可
  • 对于崩解温度低于共晶温度的产品，以控制崩解温度为准。产品的崩解温度取决于产品本身的物性和 保护剂的种类，优选具有较高崩解温度的材料，使升华干燥能在不太低的温度下进行，可以减少能耗，节约时间，提高生产效率。一般来说，塌陷温度 Tc 比共晶温度 Te 稍高，共晶温度比玻璃转化温度 Tg 高。

什么是塌陷：固体结构松散，关键要点有哪些

回答BioCloudHub

  • 冷冻干燥机得冷冻干燥现阶段普遍用以生物制药的储存。大部分生物制药降温时产生玻璃化变化，干燥全过程中试品温度过高造成坍塌，从而造成干燥层的多孔材料缺失、残留水份增加、复水时间增加，更比较严重的将造成生物制药性缺失。因而，坍塌温度在冻干全过程中是一个十分关键的主要参数。坍塌与很多要素相关，不仅有原材料自身物理性能加工参数的危害又有生产加工参数的危害。冷冻干燥机的坍塌溫度：冻干时原材料中的冰晶消退，原来为冰晶所占有的室内空间变成空化，因而冻干层呈多孔结构蜂巢状海绵体构造。此构造与温度相关。当蜂巢状建筑结构的固态栽培基质温度较高时，其刚度减少。当温度到某一临界点时，固态栽培基质的刚度不能保持蜂巢状构造，空化的固体物质栽培基质壁将产生坍塌，原来蒸汽扩散的安全通道被封闭式，此临界压力称之为冻干原材料的奔溃溫度或坍塌温度。 原因：在一次干燥或者二次干燥阶段，产品温度超过玻璃态转化温度或者塌陷温度。当储存温度接近产品玻璃态转化温度的时候，也可能在产品的存储阶段发生。而玻璃化转 变与胶塞的高残余水分共同促进了产品的坍塌。 影响：失去了蛋糕良好的外观
  • 由于塌陷导致冷冻过程中形成的内部微观结构表面积 减少
  • 残余水分含量高
  • 产品重建时间变长。 建议:在初次干燥阶段内（当产品中仍有冰时），将产品温度保持在崩溃温度以下。避免在表面顶部形成

什么是开裂：蛋糕裂开，关键要点有哪些

什么是冻融稳定性研究，关键要点有哪些

回答BioCloudHub

由于原液在低温存储，并且可能需要多次冻融，冻融过程会对蛋白质的结构和功能造成一定的影响，但影响的大小取决于多种因素，如蛋白质的种类、浓度、pH 值、温度、冻结速率、冻结温度、解冻速率等。

一般来说，蛋白质的结构和功能受到的影响越小，其稳定性就越好。

在冻融过程中，蛋白质的结构可能会发生变化，如蛋白质的二级结构（α-螺旋、β-折叠等）和三级结构（蛋白质的立体构型）可能会发生改变，从而影响其功能。

此外，冻融过程中可能会发生蛋白质的聚集和凝集，导致蛋白质失活或失去功能。

完整性测试常见有哪些类型

回答BioCloudHub

除菌滤器完整性测试类型分为破坏性测试（颗粒挑战测试，或细菌挑战测试）和非破坏性测试（包括泡点、扩散流、水侵入、压力衰减测试）。

破坏性测试一般是由供应商完成的，其目的是确定该滤器是除菌级滤器，测试方法即为 FDA 对除菌级过滤器的定义。

例如:细菌挑战试验用作破坏性检测。

在这种检测中，含有细菌的培养基溶液在过滤器的上游引入并确保在过滤器后回收的有机体为零。

这种检测被称为破坏性的，因为在这个检测中“有意”加载微生物培养物，并且之后不能再使用，即过滤器必须销毁。

非破坏性完整性测试的主要目的是在不损坏过滤器的前提下确定是否存在可能危及过滤器截留能力的缺陷，它需要供应商在出厂时及客户在使用现场进行操作，而用户一般只需要进行非破坏性测试即可，此外供应商也需进行非破坏性测试。

什么时候做完整性测试

回答BioCloudHub

  • 完整性测试一般可以选在三个时间段进行操作： o 灭菌前：查套筒是否密封好，安装是否正确，管路和配件的连接是否有问题以及滤芯本身是否有问题。 o 灭菌后，使用前：灭菌过程对完整性测试的影响； o 过滤使用后：料液的化学兼容性对滤器的影响以及操作压力的影响。

什么是完整性测试的关联，关键要点有哪些

回答BioCloudHub

所有的非破坏性完整性检测标准必须与破坏性完整性测试结果相关联才能是有效的，细菌截留试验对除菌能力是最灵敏的完整性检测试验，非破坏性的完整性试验是和细菌挑战试验关联的替代性试验。挑战实验结果与完整性检测结果关联，一般用户只需要检测完整性数据即可判断除菌级滤芯的细菌截留效率情况。

原液颜色的质量标准是什么

回答BioCloudHub

在抗体生产中，原料药（DS）的颜色也是放行指标之一，DS 的典型颜色为淡黄色或接近无色。

尽管颜色异常并不意味着药物安全性和有效性一定存在问题，但对产品纯度、生物活性、运输和存储的担忧仍然存在。

欧洲药典中规定用于临床试验的 DS 应为无色或接近无色。

因此，一旦发生 DS 颜色异常，立即启动 RootCause 调查和后续改进工艺对保证安全生产至关重要。
